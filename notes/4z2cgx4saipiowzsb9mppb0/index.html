(arXiv 2023)H2-Mapping

H2-Mapping: Real-time Dense Mapping Using Hierarchical Hybrid Representation

可以在边缘计算的设备上实时构建高质量的地图。

  • 层次混合表示。利用隐式多分辨率hash编码，描述不同层次的细节。使用了八叉树sdf先验来帮助构建隐式地图，可以加速地图初始化和学习。
  • 最大覆盖关键帧选择策略。解决遗忘问题和提高地图质量。
  • 边缘设备运行。
